**** PARA A UTILIZAÇÃO DAS MODIFICAÇÕES ****

1 - PASSO
Faça sempre backup da sua base dados e dos arquivos do GLPI.

2 - PASSO
Substitua os arquivos na raiz da sua instalação do GLPI. (Obs: Os arquivos já estão nas suas respectivas pastas para substituição)

3 - PASSO
Inserir no MySQL os dados do arquivo "insert_glpi_crontasks.sql" (Há um insert da glpi_config também).

4 - PASSO
Acesso o GLPI e veja as modificações funcionando.
